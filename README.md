# 🦴 骨科多模态临床科研项目 (Orthopedics Multi-modal AI Project)

## 📖 项目简介
本项目是一个立足于真实临床场景的**多模态骨科人工智能科研项目**。数据集深度融合了 **CT、MRI、XRay** 三大医学影像模态，以及高度结构化的临床电子病历（涵盖住院诊断、护理评分、疼痛追踪与千万级生化检验流水）和骨密度报告。

依托于这套多源异构数据底座，本项目正在逐步推进各项科研任务。目前初步规划的核心课题包括但不限于：
1. **轻量级预测任务**：利用检验数值等结构化生化指标，进行骨折风险等临床事件的快速预测建模。
2. **影像学分级与诊断**：基于纯净的骨科影像集，开展骨折严重程度分级等机器视觉任务（标注策略视医生需求而定）。
3. **术后再骨折风险综合预测**：结合骨水泥注入等临床干预手段，探索相邻椎体力学变化（如 3D 有限元分析），结合多模态特征进行术后并发症预警。

## 🚀 研发时间线与里程碑 (Timeline & Milestones)

- **[2026-04-24] 骨科有限元代理模型（FEA）架构预研与多尺度特征处理**
  - **FEA 深度学习模型架构设计**：确定了“仿生双流”混合表征网络初步架构。利用数学形态学分离“皮质骨外壳（图/点云流）”与“松质骨及骨水泥内芯（体素流）”，计划通过交叉注意力机制（Cross-Attention）进行特征融合，并在输出端通过点云体素化（Splatting）重构 3D 应力场。该架构旨在提升大样本 FEA 提取的计算效率，并保留底层力学传导的梯度特征。（*详细模型设计与文献调研见 [FEA模块预研文档](./DOCS/FEA_PRE.md)*）。
  - **多尺度重采样与 GPU 物理映射（56-59号脚本）**：完成支持 1.0x/1.25x/1.5x 缩放因子的多精度数据提取流程。利用 CuPy 进行 GPU 加速，实现基于体素密度（HU值）的杨氏模量与轴向应力场快速推演，有效缩短单样本的物理仿真耗时，并输出 3D 应力张量（HDF5/VTK）。
  - **半精度压测与 3D 可视化数据处理（50-55号脚本）**：针对临床 CT 堆叠的 3D 数据进行 FP16 半精度压缩与 HDF5 存储测试，以降低后期训练的显存占用与 I/O 压力。同时编写了 HDF5, NIfTI (.nii.gz) 及 STL 格式的转换工具链，以适配 ParaView 等医学三维可视化软件的验证需求。

- **[2026-04-17] 底层影像溯源、特征工程基建与前沿架构调研**
  - 全量影像底层溯源排查：针对医院系统初始导出数据中 CT 模态大面积缺失及少量其他模态数据对不齐的异常情况，启动了物理底层的极速全量快照扫描。通过多重交叉审计，查明了系统账单遗漏与过滤逻辑的根源，最大程度追回并夯实了真实有效的多模态影像资产。
  - 骨水泥术后再骨折风险深度调研：围绕 PVP/PKP 术后并发症，完成了基于生物力学与 AI 交叉领域的前沿课题调研。明确了融合有限元分析（FEA）应力热力图与临床生化语境特征的多模态深度学习路线（涵盖 DL-FEA 代理模型、PINNs 及 LGA 架构），为模型构建确立了高维度的理论基石。
  - 检验大宽表宏观特征普查：针对生成的稀疏特征大宽表，设计并运行了宏观维度统计探针，全面测算数百项生化指标的“总检验频次”与“独立患者覆盖率”。成功排查出因“检验套餐前缀”导致的黄金特征（如骨代谢指标）撕裂陷阱，为下一步入模特征的精准折叠与提纯提供了全局视角的数据支撑。

- **[2026-04-10] 临床表格降维与全景多模态交集闭环**
  - 完成《患者基本信息及住院记录》、《护理评分》、《疼痛评分》等非检验类临床表格的全局时序重排与基准对齐，消除系统导出乱序。
  - 成功建立涵盖核心生化指标的同义词映射大表。
  - 穿透“四级清洗漏斗”，将庞杂的千万级检验长表完美降维转置为以 `[患者ID, 报告时间]` 为主键的特征稀疏大宽表。
  - 跑通物理影像目录与 HIS 映射账单的双向交叉审计，成功输出多模态患者组合矩阵（含五维大满贯黄金队列）。

- **[2026-04-03] 影像底层解析与切片级防爆清洗**
  - 完成 DICOM 图像物理底层的元数据强制提取引擎开发，确保在面对损坏头文件或非常规后缀时的极高健壮性。
  - 成功构建并迭代基于多维语义字段（`BodyPartExamined`, `SeriesDescription`, `StudyDescription`, `ReceiveCoilName`）的动态黑白名单机制。
  - 攻克了早期的“文件夹连坐误杀”难题，实现 UID（`Image_Data_ID/Image_Name`）切片级精确扣减，精准剔除胸部/颅脑等非核心扫描，最大化保全了真实有效的脊柱影像资产。

- **[2026-03-28] 核心多模态数据初步集结**
  - 原始数据收集阶段性完成。成功获取涵盖三大影像模态（CT、MRI、XRay）的全量底层 DICOM 物理文件。
  - 同步就位各项临床记录表（含病案首页、手术记录）、护理与疼痛评估表，以及全量化验指标原始长表（除骨密度报告外，核心数据矩阵基本构建完毕）。

- **[2026-02-03] 全球脊柱影像公开基准与前沿基础模型演进调研**
  - **全景数据基准摸底**：详尽梳理了 2016-2026 年间全球顶尖脊柱挑战赛，并完成对主流公开数据集（如 Endplate3D-QCT、TotalSegmentator、CTSpine1K 等）的多维度评估，明确了利用基础模型弥合公开域与私有临床数据间“多模态鸿沟”的必要性。
  - **前沿架构落地探路**：深度解析了医学影像分析从“几何形态识别”向“多模态语义理解”的范式跨越。重点调研了 **Medical SAM3**（指令驱动分割）与 **SpineMamba**（残差视觉 Mamba 层）等前沿架构在骨科临床场景中的潜在落地形态，确立了项目的先进性基准。

- **[2026-01-28] 骨科 AI 课题立项与小样本生成式破局架构设计**
  - **临床目标确立**：基于 MICCAI、CVPR 等顶会最新动向，明确课题从单一骨折检测向“理解并发症风险机制”演进，初步设计了融合微观骨代谢标志物（P1NP、$\beta$-CTX）与宏观影像学的多模态智能诊疗框架。
  - **核心技术路线定型**：针对初期面临的“小样本（13例）与非标准影像”极限痛点，果断摒弃传统全监督路线，确立了“基础模型 + 参数高效微调（PEFT）”的战略。
  - **前瞻性架构构思**：提出了基于 **Language Guide Adapter (LGA)** 的文本引导架构，并引入 **HealthiVert-GAN**（伪健康椎体合成）与 **Blaze3DM**（低质截图三维恢复）等生成式 AI 技术，作为攻克历史劣质资产的底层技术储备。
